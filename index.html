LocalSports

LocalSports is an Android application to allow users follow sport competitions from their towns.
Functionalities offered by the app:

  • Check matches calendar dates, places and teams, also the final score when match is already played.

  • Check classification status: the points, matches played, won, lost and drawn.

  • Follow competitions, so that user will receive notifications when a update happend in this compotitions. Also it is possible to disable notifications.

  • Follow team, unlike follow competitions in this case user only will receive notifications when an update has an impact on the team

  • Notify error: when user detect that some information is not correct can send a message to the administrator of the system in order to correct the data.

  • Calendar integration: it is possible to add an event of the match date and time to the device calendar.

  • Share match info: user can share info of the match like date, place, score by Whatsapp, email or sms.

  • Map integration: it is possible to open the device map, to see where the match will take placed.
